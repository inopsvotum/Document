# Mobile Game Marketing Plan — Magic Pylon (Action Roguelite)

## 1) Positioning (what we’re selling)
**Core hook:** Fast 2–4 minute runs + skill-based ability combat + defend-the-pylon tension.

**Primary audience:**
- Hybrid-casual action players
- Survivors-like / roguelite players
- Competitive short-session players (leaderboard mindset)

**Message pillars:**
1. Fast runs (fits commute/micro-breaks)
2. High skill expression (aim-slow + ability builds)
3. Replayability (different heroes/builds each run)

---

## 2) Go-to-market phases

### Phase A — Pre-launch (4–6 weeks)
- Create 8–12 ad creatives:
  - 3 gameplay truth videos (real in-game)
  - 3 fail-vs-clutch short clips
  - 3 hero fantasy clips
- Build store assets:
  - 2 icon variants
  - 2–3 screenshot sets
  - 2 app-store descriptions
- Set up tracking events:
  - install, tutorial_complete, run_start, run_complete, day1_return, purchase
- Soft-launch in 1–2 geo markets first.

### Phase B — Soft launch (4 weeks)
- KPI focus:
  - D1 retention, tutorial completion, run completion rate, CPI
- Weekly optimization loop:
  - Cut bottom 30% creatives
  - Duplicate top 20% with variants (first 3 seconds changed)
- Run small budget across Meta + TikTok + Google UAC.
- Add creator seeding (micro creators with roguelite audiences).

### Phase C — Global launch (6–8 weeks)
- Scale only if retention thresholds are met.
- Add live events + season pass at/near launch week.
- Use leaderboard/event hooks in ads (e.g., “Can you survive Wave 12?”).

---

## 3) Channel strategy

1. **Paid UA (core):** Meta, TikTok, Google UAC (creative-led optimization)
2. **Influencer/micro-creator:** 10k–200k creators in mobile gaming niches
3. **Community flywheel:** Discord + TikTok Shorts + YouTube Shorts
4. **ASO:** continuous icon/screenshot/title tests

---

## 4) Creative playbook

Top angles to test:
- Last 10% pylon HP clutch
- Hero fantasy burst (Berserker wipe / Mage nuke)
- Build evolution in 20 seconds
- “Can you beat my score?” social challenge

Creative structure (15–25s):
1. Hook in first 1.5s
2. Show problem (wave pressure)
3. Show mechanic (aim-slow, ability combo)
4. Show payoff (clear + score)
5. CTA (Play free now)

---

## 5) KPI gates (ship/no-ship)

- **D1:** 35%+ (good), 40%+ (strong)
- **D7:** 10–15%+
- **D30:** 3–6%+
- **Tutorial completion:** 80%+
- **First-run completion:** 65%+
- **Ad CTR:** 1.5%+ baseline, 2%+ strong

If below threshold:
- fix onboarding friction
- reduce early fail spikes
- improve first power spike in first 90 seconds

---

## 6) Monetization + marketing alignment

- starter pack
- low-price revive bundle
- rewarded ad loop (extra buff/continue)
- battle pass + timed events in launch campaigns

Marketing promise should match actual gameplay.

---

## 7) 90-day execution plan

- **Days 1–30:** retention + onboarding tuning
- **Days 31–60:** scale winning creatives + creator program
- **Days 61–90:** global push + seasonal cadence + lookalikes from high-value users

---

## Notes
This plan is execution-focused and should be paired with weekly KPI review and creative refresh cycles.
